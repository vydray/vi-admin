# 申請

各種申請を出す画面です。アプリ右上の **「⋯ メニュー → 申請」** から入れます。

## 用途

シフトアプリの「申請」機能は、シフト以外の **管理者への連絡・依頼** をまとめる窓口です。

### 申請できる主な内容

| 種類 | 用途 |
|---|---|
| 給与に関する問い合わせ | 給与明細の数字を確認したい時 |
| 商品追加リクエスト | こんな商品を入れてほしい等 |
| 勤怠申し送り | 当日連絡（遅刻、体調不良 等） |
| その他 | 上記に当てはまらない要望 |

## よく使う操作

### 新規申請を出す

1. 申請画面で **「+ 新規申請」** ボタン
2. 種類を選択
3. 内容を入力
4. 必要に応じて画像を添付
5. **「送信」** ボタン

### 申請の状態を確認する

申請一覧で各項目の状態が確認できます。

| 状態 | 意味 |
|---|---|
| 申請中 | まだ管理者が確認していない |
| 承認 | 管理者が承認した |
| 却下 | 管理者が却下した（理由が表示される場合あり） |
| 完了 | 対応完了 |

### 過去の申請履歴を見る

申請一覧をスクロールすると過去のやり取りも確認可能。
- 承認/却下の通知は本アプリのベル（🔔）にも届きます

> 💡 緊急の連絡は申請機能より直接管理者の LINE / 電話で行ってください。申請は記録として残す用途に向いてます。
