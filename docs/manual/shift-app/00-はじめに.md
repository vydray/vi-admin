# はじめに

シフトアプリは、キャスト向けの LINE ミニアプリです。シフト申請・確認、売上ランキング、給与明細などをスマホから確認できます。

## 利用方法

### 入り方

1. お店の **LINE 公式アカウント** を友だち追加
2. メニューの「**シフトアプリ**」をタップ
3. LINE 内でアプリが開きます（初回は LINE のアカウント情報利用に許可が必要）

### 動作環境

- **スマホ専用**（iPhone / Android）
- LINE アプリ内でのみ正常動作（PC ブラウザでは閲覧モードのみ）
- インターネット接続必須

## 画面構成

シフトアプリは下部ナビゲーションで 4 つの主要画面に切り替えられます。

| アイコン | 画面 | 内容 |
|---|---|---|
| 📅 シフト | [シフト](01-シフト.md) | シフト申請・確認、月間カレンダー |
| 📊 ランキング | [ランキング](02-ランキング.md) | キャスト売上ランキング |
| 💰 売上 | [売上](03-売上.md) | 自分の月間売上明細 |
| 📄 報酬 | [報酬](04-報酬.md) | 給与明細 |

その他、画面右上の「⋯」メニューから:
- **設定**: プロフィール写真、Twitter / Instagram、自己紹介 等

## トラブルシューティング

### 画面が真っ白で進まない

LINE 内ブラウザのキャッシュが原因の可能性。LINE アプリを一度終了して再起動するか、トーク画面からアプリを開き直してみてください。

### 「データの読み込みに失敗しました」

通信エラー or サーバーメンテナンス中の可能性。少し時間を置いて再度開いてください。

### ログイン情報がおかしい（別の人として表示される）

LINE アカウントが切り替わっている可能性があります。お店の管理者に連絡してください。
