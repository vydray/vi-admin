## 使い方

- 各セクションは独立しています。必要な箇所だけ読めばOK
- 画像は `images/` フォルダにまとめて入ってます
- Markdown ビューア（VS Code、Typora 等）で開くと見やすい
- Word に変換したい場合は Pandoc 推奨: `pandoc 00-はじめに.md -o 00-はじめに.docx`
