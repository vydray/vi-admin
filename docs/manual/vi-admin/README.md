## 使い方

- 各セクションは独立しています。必要な箇所だけ読めばOK
- 画像は `images/` フォルダにまとめて入ってます
- Markdown ビューア（VS Code、Typora 等）で開くと見やすい
- VS Code なら **`Cmd+Shift+V`** でプレビュー表示
- Word に変換したい場合は Pandoc 推奨:

  ```bash
  cd docs/manual/vi-admin
  pandoc README.md 00-*.md 01-*.md 02-*.md 03-*.md 04-*.md 05-*.md 06-*.md 07-*.md 08-*.md 09-*.md 10-*.md 11-*.md 12-*.md -o VI-Admin-マニュアル.docx
  ```

## スクリーンショットの数字について

画像内の数字は実データを伏せるため `███` でマスクされています。実画面では実際の数値が表示されます。
店舗名も `○○店` に置換されています。
